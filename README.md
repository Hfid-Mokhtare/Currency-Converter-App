# My Application 3

A simple currency converter application for Android.

## Features

*   Convert between different currencies.
*   Real-time conversion as you type.

## Screenshot

![currency converter](appScreenShot/Screenshot_20251230_120319.png)

## How to use

1.  Enter the amount you want to convert.
2.  Select the currency to convert from.
3.  Select the currency to convert to.
4.  The result will be displayed automatically.
