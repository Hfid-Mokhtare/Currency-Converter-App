# My Application 3

A sleek and efficient currency converter application for Android.

## Features

* **Real-time Conversion:** Results update instantly as you type.
* **Smart Selection:** Uses custom listeners to track currency pairs.
* **Data Security:** Implements hashing for internal action logging.

## How to use

1. Enter the amount you want to convert.
2. Select the currency to convert from.
3. Select the currency to convert to.
4. The result will be displayed automatically.
